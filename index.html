Serviços

Web Design

No Web Design, combino criatividade e técnica para criar interfaces visuais atraentes e funcionais. Meu foco é proporcionar uma experiência de usuário intuitiva e agradável, utilizando as melhores práticas de design responsivo e acessibilidade.

Frontend Developer

No Desenvolvimento Frontend, sou proficiente em HTML, CSS e JavaScript, além de frameworks modernos como React e Java Spring. Minha habilidade em traduzir designs em código limpo e eficiente garante que as interfaces sejam dinâmicas, interativas e perfeitamente integradas.

Backend Developer

No Desenvolvimento Backend, tenho experiência com linguagens e frameworks como PHP e Delphi XE. Trabalho com bancos de dados relacionais e não relacionais, como MySQL, PHPMyAdmin e DBeaver, assegurando que a lógica de negócio e as operações de dados sejam robustas e escaláveis.

Software Developer

No Desenvolvimento de Software, aplico princípios de engenharia de software para projetar, desenvolver e manter aplicações de alta qualidade. Tenho experiência com metodologias ágeis e controle de versão (Git), garantindo um ciclo de desenvolvimento eficiente e colaborativo.
